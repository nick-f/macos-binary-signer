# macOS Binary Signer

Helper used to generate package installers of shell scripts and other binaries

## One-time setup

### fzf

```
brew install fzf
```

### Packages

Install Packages from the [Packages
homepage](http://s.sudre.free.fr/Software/Packages/about.html) or, if you're on
macOS Ventura or later, use the latest dev version:
http://s.sudre.free.fr/files/Packages_1211_dev.dmg

See https://github.com/packagesdev/packages/issues/127#issuecomment-1341591718
